## Description

Backend do projeto Sistema de autoatendimento de fast food

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start:dev
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```

## Swagger

```bash
# run the project
$ yarn start:dev

# run the command
$ http://localhost:3000/api/docs
```

## Documentation

```bash
# To generate documentation, run the command
$ yarn doc
```

## Health Check

```bash
# run the command
$ yarn start:dev
$ http://localhost:3000/health
```
